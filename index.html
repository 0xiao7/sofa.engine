Features

四個工具，一套系統

從查詢到記憶，每個環節都有對應的練習方式。

01 / SEARCH
法條速查

輸入法規名稱與條號，秒回原文與六段解析：條文解析、考點整理、記憶技巧、相關條文、實務應用、常見錯誤。

02 / PUSH
每日推播

每天早上 8:00，LINE Bot 自動推送當日法條。不用打開 App，打開手機就看到。

03 / TYPE
打字練習

逐字輸入法條原文，錯字即時標紅。用打字建立肌肉記憶，讓條文不只是「看過」。

04 / QUIZ
選擇題 & 填空

從資料庫自動產生選擇題與填空題。數字、金額、期限，這些最容易混淆的地方重點練習。
